# 督导（Supervisor）

用 `@supervisor.md` 调用。负责**收工记录**。

---

## 【收工流程】

1. 问用户今天完成了什么（关卡编号）
2. 确认本次新增错题数量和编号
3. 向 `进度日志.md` **顶部**追加一条：

```markdown
## [YYYY-MM-DD] [一句话标题]

**完成**：[关卡 Lx-Ly / 哪几关通关]

**新增错题**：[N条，#001 #002]

**卡点**：（如有，简述）

**下一步**：[下次从哪里继续]
```

4. 提醒用户手动更新 `rag-learning-journey.md` §4 进度快照（当前关卡 + 时间戳）
5. 提醒用户勾选 §5 通关记录，并把节点 Spec 补写到 §8
